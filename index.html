Projects

My Optimum

My Optimum is a native iOS application which is related to internet troubleshooting, gateway setup, TV setup and billing. The application comprises of self service features like network troubleshooting, manually setting up the internet gateway. Also user can customise the internet connectivity as per the wish list and can also perform speed test for internet connectivity. From this application user can manage their billing operation like payments, managing auto pay setup and adding payment methods (credit card and bank).

Syncup Tracker

SyncUP Tracker is a native iOS application which is used as a tracking application with the help of IOT module. The application comprises of location tracking, SOS, Boundary controlling and Bluetooth connectivity. The application has the ability to sync with the GPS device to locate for enabling sound buzzers, tracking and SOS features. The application has the ability to notify the user whenever a boundary breach happens on the GPS device.

Safe Cab (Ionic)

Safe cab is a cross platform mobile application which is used to track the vehicle locations using the Map. This is application will be supporting both Android and iOS versions. It has the option to book a cab from the application itself. The data would be stored on AWS server for platting graphs to get the statistics, So based on the travel, the workers would get the payment as per the company rules. Also has a attendance verification option for the drivers who uses this application.

Nissan Connect Box

Nissan Connect Box is iPhone application which is developed for App store. The application is being used by car owners who has the Nissan cars, and the application would provide the metadata of the car with the analysis reports via charts. Also, it has the map view which provides the navigation option whenever the car requires an emergency service. The application dashboard would have the options to show the car parts status like tyre pressure, fuel consumption, distance travelled etc.

Rlink

Rlink is an iPhone application which is being developed for app store. The application is basically comprises of IOT functionalities and it will connect the device which would be placed on the car with the help of MQTT Broker. The application would connect to the device with help of common WIFI connection and will send the instructions to the device, according to the instructions, it would give the commands to the car devices.
